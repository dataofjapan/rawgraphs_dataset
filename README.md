# RawGraphsデータセット

RawGraphsに含まれるデータセットの変数名（ラベル）とデータ値をできるだけ日本語化しました。

- https://github.com/rawgraphs/rawgraphs-app (Version 2.0)
- https://github.com/rawgraphs/rawgraphs-charts (Version 1.0)
